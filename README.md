
# Unstop_assignment

This application is a seat booking application. There are 80 seats in total, with 7 seats in a row and 3 in the last row. Users can book at most 7 seats at a time.



## Tech Stack
- Frontend
  - Next.Js
  - HTML
  - CSS
  - JavaScript
  - Tailwind
- Backend
  - Node.JS
  - Express.Js
  - Mongoose
  - MongoDB-Atlas
